# AI in Business & Economic Research

A curated knowledge base on using AI tools and workflows for business and economic research. Built from 129+ web articles, academic papers, and practitioner guides.

**Live site:** [velikov-mihail.github.io/ai-econ-wiki](https://velikov-mihail.github.io/ai-econ-wiki/)

## Topics Covered

- **Foundations & Setup** — Installation, privacy, getting started with AI coding tools
- **Prompt Engineering & Workflow** — Prompt design, plan-driven development, workflow architecture
- **AI Agents & Agentic AI** — Agent frameworks, agentic workflows, autonomous research
- **Claude Code Skills & Advanced** — Skills, DAAF, continuous improvement, compilation workflows
- **Data Analysis & Web Scraping** — WRDS, EDGAR, scraping, data pipelines
- **Academic Research & Publishing** — Vibe research, feedback machines, AI-assisted writing
- **Finance & Econometrics** — Project APE, automated research in finance, policy evaluation
- **AI Tools & Comparisons** — ChatGPT vs Claude, Cursor, NotebookLM, tool selection
- **Institutional & Societal** — AI adoption, the bitter lesson, AI as normal technology
- **Professional Productivity** — Executive assistant, project management, tax, meetings

## Structure

```
raw/           — Source articles, papers, and images
wiki/          — Compiled wiki (summaries, concepts, visualizations)
presentations/ — Slide decks (Smeal AI Community of Practice in Research talk)
output/        — Query results, charts
```
